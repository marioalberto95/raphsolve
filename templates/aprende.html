Aprende Newton-Raphson

Comprende el método para una variable y para sistemas no lineales de varias variables.

¿Qué es Newton-Raphson?

Newton-Raphson es un método numérico iterativo que permite encontrar soluciones aproximadas de ecuaciones no lineales.

En una variable se busca un valor de \(x\) que cumpla:

\[ f(x)=0 \]

En varias variables se busca un vector \(X\) que convierta en cero todas las ecuaciones del sistema:

\[ F(X) = \begin{bmatrix} f_1(X) \\ f_2(X) \\ \vdots \\ f_n(X) \end{bmatrix} = \begin{bmatrix} 0 \\ 0 \\ \vdots \\ 0 \end{bmatrix} \]

Newton-Raphson de una variable

Se utiliza para resolver una ecuación \(f(x)=0\).

1D

A partir de una aproximación inicial \(x_0\), el método construye nuevas aproximaciones mediante:

\[ x_{n+1} = x_n - \frac{f(x_n)}{f'(x_n)} \]
1

Elegir \(x_0\)

Se selecciona una aproximación inicial cercana a la raíz esperada.

2

Evaluar

Se calculan \(f(x_n)\) y \(f'(x_n)\).

3

Actualizar

Se obtiene \(x_{n+1}\) aplicando la fórmula.

4

Comprobar

Se repite hasta que el error sea menor que la tolerancia.

Newton-Raphson para sistemas

Se utiliza cuando hay dos o más ecuaciones y el mismo número de variables.

ND

Para un sistema de \(n\) variables se trabaja con un vector de aproximaciones:

\[ X_n = \begin{bmatrix} x_{1,n} \\ x_{2,n} \\ \vdots \\ x_{n,n} \end{bmatrix} \]

Las derivadas parciales se organizan en la matriz Jacobiana:

\[ J(X) = \begin{bmatrix} \frac{\partial f_1}{\partial x_1} & \frac{\partial f_1}{\partial x_2} & \cdots & \frac{\partial f_1}{\partial x_n} \\ \frac{\partial f_2}{\partial x_1} & \frac{\partial f_2}{\partial x_2} & \cdots & \frac{\partial f_2}{\partial x_n} \\ \vdots & \vdots & \ddots & \vdots \\ \frac{\partial f_n}{\partial x_1} & \frac{\partial f_n}{\partial x_2} & \cdots & \frac{\partial f_n}{\partial x_n} \end{bmatrix} \]

En cada iteración se resuelve el sistema lineal:

\[ J(X_n)\Delta X_n=F(X_n) \]

Después se actualiza el vector:

\[ X_{n+1}=X_n-\Delta X_n \]
RaphSolve resuelve el sistema lineal directamente. No calcula la inversa de la Jacobiana, lo cual mejora la estabilidad numérica.

Procedimiento para sistemas

1

Definir el sistema

Se ingresa una ecuación por cada variable.

2

Elegir \(X_0\)

Se proporciona un valor inicial para cada variable.

3

Evaluar \(F(X_n)\)

Se calcula el valor de cada ecuación en la aproximación actual.

4

Evaluar \(J(X_n)\)

Se calculan todas las derivadas parciales.

5

Obtener \(\Delta X_n\)

Se resuelve el sistema lineal definido por la Jacobiana.

6

Comprobar convergencia

Se revisan el error aproximado y la norma del residuo.

Ejemplo de dos variables

Consideremos el sistema:

\[ \begin{cases} x^2+y^2-4=0 \\ x-y=0 \end{cases} \]

Como \(x=y\), la solución positiva se encuentra cerca de:

\[ \begin{aligned} x &\approx 1.41421356, \\ y &\approx 1.41421356 \end{aligned} \]

En este caso se pueden representar las dos superficies en una gráfica 3D y observar la trayectoria de las aproximaciones.

Ejemplo de tres variables

\[ \begin{cases} x^2+y+z-3=0 \\ x+y^2+z-3=0 \\ x+y+z^2-3=0 \end{cases} \]

Una solución del sistema es:

\[ \begin{aligned} x &= 1, \\ y &= 1, \\ z &= 1 \end{aligned} \]

Para tres variables, RaphSolve representa la trayectoria de las aproximaciones en el espacio.

Error y residuo

En sistemas de varias variables se utilizan dos indicadores para decidir si el método convergió.

Error aproximado

\[ \lVert X_{n+1}-X_n \rVert_{\infty} \]

Mide cuánto cambió la nueva aproximación.

Norma del residuo

\[ \lVert F(X_{n+1}) \rVert_{\infty} \]

Mide qué tan cerca están las ecuaciones del vector cero.

Ventajas

  • Puede converger muy rápidamente cerca de una solución.
  • Funciona con una variable y con sistemas no lineales.
  • Permite obtener resultados con alta precisión.
  • La Jacobiana se puede calcular automáticamente.

Limitaciones

  • El resultado depende de los valores iniciales.
  • Puede fallar cuando la derivada o la Jacobiana son cero o singulares.
  • No garantiza convergencia para todos los problemas.
  • Para sistemas grandes, la interpretación gráfica se limita a proyecciones de tres variables.

Recomendaciones

Valores iniciales

Utiliza aproximaciones razonablemente cercanas a la solución esperada.

Tolerancia

Una tolerancia pequeña mejora la precisión, pero puede requerir más iteraciones.

Validación

Comprueba que la solución haga que las funciones sean cercanas a cero.
